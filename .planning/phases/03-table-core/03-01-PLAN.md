- Model [colspan 1, no group-start] | Spec [colspan 3, group-start left-border] | Score [colspan benchmarks*2, group-start left-border]

- Empty cells for Model and Spec columns (for visual alignment)
- For each benchmark: label cell spanning 2 columns (Acc + Time), with group-start left-border on first benchmark

*Row 3 (leaf-row, sortable):*
- Model [no-sort class, no click handler] | Params | Quant | Size | [Per benchmark: 🎯 Acc | ⏲ Time ]

- All leaf cells except Model should have `data-col` attribute (e.g., `data-col="spec.parameters_b"`, `data-col="scores.MMLU.accuracy"`)

- Apply CSS classes: `group-row` on rows 1-2, `leaf-row` on row 3, `group-start` on appropriate th elements
- No colspan attributes where colspan="1" (omit for cleaner HTML)
- Keep existing table styling; add new classes for three-tier structure
