**Create src/composables/useBenchmarkData.ts:**

Implement a reactive composable that watches a device Ref and fetches benchmark data on change.

Function signature: `useBenchmarkData(device: Ref<string | null>)`

Implementation:
1. Create refs for reactive state: `entries: Ref<Entry[]>`, `isLoading: Ref<boolean>`, `error: Ref<string | null>`
2. Define an async function fetchData(deviceKey: string) that:
   - Sets isLoading to true
   - Calls fetch(`/data/${deviceKey}.json`)
   - Parses response as JSON
   - Validates that response is an array of Entry objects (or accepts as-is if field structure matches)
   - Assigns to entries ref
   - Sets isLoading to false
   - On error: catches error, stores error.message in error ref, logs to console, does not re-throw
3. Use watch(device) to call fetchData whenever device changes, but only if device is non-null
4. Return object: `{ entries, isLoading, error }`

Error handling: if fetch fails or JSON parse fails, set error.value and isLoading.value to false.

Guard: `if (typeof window !== 'undefined')` wrapper around watch (defensive coding for SSG safety).

Import Entry from '../types/benchmark'. Import ref, watch, onMounted from 'vue'.
